# Beaker Theme
Beaker is the new Amazee Labs Standard Theme

## Gulp
Beaker is using Gulp. We use gulp to compile SASS and run Browersync.
to run gulp do the following steps:

###Install npm
npm install

### Run gulp
gulp watch
